#  Cine Las Palmas – Proyecto Web

Este proyecto es una página web completa creada para simular el sitio oficial del **Cine Las Palmas**, situado en **Las Palmas de Gran Canaria**.  
Incluye varias páginas, un diseño responsive, una galería, un formulario validado en JavaScript, un CRUD básico y un efecto parallax.

---

##  Descripción del Proyecto

El objetivo del proyecto es desarrollar una página web moderna utilizando únicamente:

- **HTML**
- **CSS**
- **JavaScript**

Además:

- Es responsive para **móvil, tablet y ordenador**.
- Contiene una galería con 10 imágenes reales de películas.
- Incluye un **CRUD (Create + Read)** para gestionar películas de la cartelera.
- El formulario está validado **solo con JavaScript**.
- El estilo está centralizado en un solo archivo CSS.
- Se ha utilizado una única librería únicamente para los iconos.
- Contiene un efecto **Parallax** alineado con la temática del cine. ( o lo intente aunque sea)
